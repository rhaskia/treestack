`+` plus
`-` minus
`/` divide
`*` multiply

`[` enter stack
`]` exit stack

`.` output as number
`,` output as char

`++` plus one
`--` minus one

`!` not symbol, treats number as a bool
`>`, `<`, `<=`, `>=` regular compare operaters
`=` equals
 
# Requires a pointer name
`&` creates a pointer if followed with a name
`*` pushes the value at the pointer to the current stack
`^` jumps to the pointer
